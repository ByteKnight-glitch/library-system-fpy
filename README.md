# library-system-fpy
--------------------------------------------------------------------------------------

Sistema de Gestión de Biblioteca (Modular) - FPY1101

Solución para la Evaluación Parcial 4 de Fundamentos de Programación. El proyecto está estructurado de forma modular para separar la interfaz de usuario, las validaciones y la lógica de negocio.

--Estructura del Proyecto--

* **`Ejercicio 1.py`**: El punto de entrada del programa. Contiene el ciclo principal (`while`) y coordina las llamadas a los diferentes módulos.
* **`menu.py`**: Contiene las funciones encargadas de mostrar el menú interactivo y capturar/validar la opción ingresada por el usuario.
* **`funciones.py`**: Alberga toda la lógica de almacenamiento (CRUD) en la lista de diccionarios, así como las funciones de validación de campos (título, copias, préstamos y disponibilidad).

# Cómo ejecutar #
Para iniciar el programa, ejecuta el archivo principal desde tu terminal:
python "Ejercicio 1.py"
